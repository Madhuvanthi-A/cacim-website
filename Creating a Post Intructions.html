Creating a Post Intructions

Place your cursor at the top left if you want to add in an image then click ADD MEDIA button top left above the editing TOOL BAR. Once the image is in the content area you can click on it to edit, change alignment, wrap text around it or link it.

You can hyperlink by bringing your cursor over the text and clicking on the LINK ICON in the first line of the TOOL BAR. You can unlink by clicking on the BROKEN LINK ICON.

You will stay in the VISUAL EDITOR MODE (Tabs at the Top Right Corner including TEXT, CORNERSTONE). If you have to interact with the HTML code for some reason click on TEXT.

If you are missing the 2nd line of your EDITING TOOL BAR click on the last icon in the first row, it will open both rows up.

You can SAVE DRAFT or PUBLISH when you are ready. You can also schedule when the post will come out by clicking on PUBLISH ON and select a date. And you can create a PASSWORD PROTECTED PAGE be selecting this option under VISIBILITY

You will not be interacting with FORMAT box.
